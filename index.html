Features of Visual Studio Code

  • Editing: Fast editor with syntax highlighting, bracket-matching, auto-indentation, and more
  • Developer tooling: IntelliSense code completion, debugging, semantic code understanding and navigation, and code refactoring
  • Version control: Built-in support for git and many other source control providers
  • Testing: Run automated tests and view test coverage to validate your code
  • Code anywhere: Use a container, remote machine, or WSL as your development environment
  • Language support: Supports hundreds of languages
  • Split view: Work on two different files of the same project simultaneously
  • Keyboard shortcuts: the keyboard shortcuts in the Keyboard Shortcuts editor
  • Configure allowed extensions: which extensions can be installed in VS Code
  • Status bar:See errors and warnings

Shorcut Keys and Their Meanings

Shortcut key Meaning
Ctrl + C: Used to copy the selected text or item to the clipboard.
Ctrl + V: Used to paste the copied content from the clipboard.
Ctrl + X: Used to cut the selected text or item (removes it and places it on the clipboard).
Ctrl + Z: Used to undo the last action.
Ctrl + Y: Used to redo the last undone action.
Ctrl + S: Used to save the current document or file.
Alt + Tab: Used to switch between open applications.
Ctrl + P: Used to open the print dialog for printing the current document or page.
